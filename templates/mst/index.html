About Minimum Spanning Tree

A Minimum Spanning Tree (MST) is a subset of edges in a weighted, connected graph that connects all vertices with the minimum total edge weight, without forming cycles.


Prim's Algorithm:

  1. Start with an arbitrary vertex
  2. Repeatedly add the minimum weight edge that connects a vertex in the tree to one outside
  3. Continue until all vertices are included

MST is widely used in network design, circuit design, and clustering algorithms.
